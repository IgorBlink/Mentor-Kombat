### 🚀 Установка

```bash
# Клонируй этот шедевр
git clone https://github.com/your-username/mentor-kombat.git
cd mentor-kombat

# Установи зависимости (молись, чтобы не конфликтовали)
npm install
# или если ты крутой
pnpm install

# Запусти сервер разработки
npm run dev

# Открой http://localhost:3000 и готовься к славе
```
